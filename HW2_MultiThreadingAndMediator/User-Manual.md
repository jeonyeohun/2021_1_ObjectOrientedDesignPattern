# User Manual

## Prerequsite

- java 1.8 or higher

## Compilation

```bash
javac Main.java
```

## Run

```bash
java Main
```

## How to use

1. 자동차의 초기 속도를 입력합니다. 이 속도를 기준으로 증속, 감속 정도가 결정됩니다.
2. 입력후 Enter 를 누르면 시뮬레이션이 시작됩니다.
3. Blackbox 결과 출력이 시뮬레이션이 완전히 종료되기 전에 출력될 수 있습니다. 최상위 우선순위를 가진 쓰레드가 종료되면 곧바로 결과를 출력합니다.
