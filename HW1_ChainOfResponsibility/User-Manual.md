# User Manual

## Prerequsite

- java 1.8 or higher

## Compilation

```bash
javac Main.java
```

## Run

```bash
java Main
```

## How to use

1. 연산자를 입력합니다. 연산자는 다음 키워드들 중 하나로 입력할 수 있습니다.

   - add : 더하기 연산
   - sub : 빼기 연산
   - mult : 곱하기 연산
   - div : 나누기 연산

2. 첫번째 피연산자를 입력합니다. 입력할 수 있는 값의 범위는 java의 integer 범위입니다.

3. 두번째 피연산자를 입력합니다. 입력할 수 있는 값의 범위는 java의 integer 범위입니다.
